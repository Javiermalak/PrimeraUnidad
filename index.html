Definiciones de Hipermedia/Hiperdefiniciones

Hiper
La RAE lo define como “exceso” o “grado superior a lo normal”
Hipertexto
Herramienta informática que permite presentar documentos o información de forma estructurada estableciendo vínculos entre estos (hipervínculos o enlaces) de manera que el usuario puede acceder a la información de forma no secuencial.
Multimedia
Tecnología que utiliza distintos medios como audio, video, imágenes o texto para transmitir información y que tiene diferentes propósitos como entretener o educar.

Hipermedia = multimedia + hipertexto

Tecnología que utiliza relaciones asociativas entre la información contenida dentro de datos multimedia con el propósito de facilitar el acceso y manipulación de la información encapsulada en los datos de forma no secuencial.

¿Algunos ejemplos de Hipermedia?

  • Enciclopedias digitales (Encarta, Wiki)
  • DVD de Películas
  • Presentaciones (Power Point, Flash)

La Hipermedia es No Lineal

Tipos de Enlaces

Estructurales
se utilizan para proveer organización en las piezas de información (Índice).
Asociativos
para establecer relaciones semánticas entre diferentes elementos (Referenecias a elementos relacionados).
Referenciales
para elaborar sobre un tópico (explicaciones, definiciones)

Hipermedia + Aprendizaje

Los programas educativos hipermedia se distribuyen a través de medios de almacenamiento portátil o la web.

Aplicaciones de Hipermedia pueden contener libros de texto completos y enciclopedias que se pueden complementar con elementos multimedia.

Riesgos

  • Una mala estructura dificultaría su navegación
  • Con una difusión inadecuada se perderían entre todos los contenidos de la web

El termino Hipertexto está asociado a otros conceptos con el prefiho “Hiper”.

Hipertexto

Retomando: Para la RAE el prefijo “Hiper” significa superioridad o exceso.

En “Hipertexto” aplican ambos significados:

  • El Hipertexto contiene una gran cantidad de textos (Exceso)

    • Wiki
    • Encarta
  • El hipertexto es también un conjunto de documentos conectados entre sí de tal manera que forman una estructura que permite al usuario ir de un documento a otro (superioridad/ir más allá)

Ted Nelson (Xanadú)

  • Acuña el termino Hipertexto en "A Conceptual framework for man-machine everything“
  • Define el prefijo “Hiper” como “ampliado, generalizado y multidimensional”
  • Texto deriva del latín textum o textus, derivado de texére tejer
  • Nelson entonces define Hipertexto como un “inmenso tejido o un enorme entramado de textos que van más allá del propio texto enlazado”

Hipertexto

Se ha incluido en el termino Hipertexto a una gran cantidad de entidades

  • Páginas Web
  • Enciclopedias Digitales
  • Programas muy populares (Ayuda de Windows)
  • Documentos creados en un procesador de textos que permita establecer enlaces

Todas estas entidades tienen en común la manera en que organizan la información

  • El hipertexto tiene una estructura no lineal.
  • Este tipo de estructura permite ir de un punto a otro en el texto o en otro texto a través de enlaces, también conocidos como hipervínculos. Gracias a los hipervínculos los usuarios pueden acceder si así lo desean a la información que buscan directamente.
  • Un hipertexto se puede construir únicamente en una computadora.

Hipertexto (Confusión)

Hipertexto:
modelo teórico de organizar la información de manera no secuencial.
Hipertexto no es:
herramienta de software o programa informático que permite leer y escribir documentos hipertextuales.
Hipertexto no es:
Documento digital que permite acceso no secuencial a la información.

Hipertexto (Clarificación)

modelo teórico de organizar la información de manera no secuencial.
Sistema de gestión de hipertextos:
herramienta de software o programa informático que permite leer y escribir documentos hipertextuales.
Hipertdocumento:
Documento digital que permite acceso no secuencial a la información.

Multimedia

Es la combinación de dos o más medios de forma simultanea.

El concepto multimedia no es privativo de la informática.

  • Un libro acompañado de una pista de audio es multimedia.
  • Sin embargo, con el tiempo el termino se ha ido generalizando para ser empleado en el mundo digital.

Hipermedia

La Hipermedia es la conjunción de Hipertexto + Multimedia

  • Multimedia contribuye a enriquecer la información presentada.
  • Hipertexto provee una estructura que permite al usuario acceder a la información de forma no lineal.

En resumen:

Hipertexto
texto en formato no secuencial, compuesto de nodos y enlaces que los interconectan.
Multimedia
Multimedia: unión de diferentes medios o morfologías de la información, como texto, gráficos, audio, vídeo, otros recursos audiovisuales, etc.
Hipermedia
hipertexto + multimedia

Del texto al hipertexto

El medio de almacenamiento que mayor aumento ha tenido ha sido internet y otros medios digitales

El texto impreso (libros, folletos, revistas, periódicos, etc.) es el medio de transmisión de información y cultura por excelencia.

  • Sigue vigente
  • La producción editorial se incremente anualmente
  • El almacenamiento impreso de información también aumenta anualmente

Del papel a la pantalla

El hipertexto impuso la pantalla como el principal medio de soporte de visualización.

El papel deja de ser el medio de soporte privilegiado.

La pantalla no reemplaza al papel, solo representa una nueva forma de mostrar información

  • Es muy compatible con el hipertexto porque favorece la presentación multi-secuencial de la información que es característica principal del hipertexto.
  • El libro (impreso en papel) continúa vigente; con el hipertexto, solo se crean nuevos soportes, formatos de almacenamiento, nuevas funcionalidades.

La escritura

De nuevo, la aparición del Hipertexto supone un cambio.

  • Hasta antes del hipertexto la escritura había sido secuencial.
  • Inicialmente los libros y otros textos eran reproducidos a mano con la ayuda de “copistas”. La llegada de la imprenta cambió esto logrando la reproducción de textos e imágenes a gran escala y de forma mecánica.
  • Con la llegada de la PC tanto la escritura como la edición y la reproducción de textos y otras obras se incrementó extraordinariamente ya que es una tecnología accesible a la mayoría de las personas y facilita su distribución.

XML Origen y Definición

XML es un lenguaje de marcado.
eXtensible Markup Language (XML) describe una clase de objetos de datos llamados documentos XML y de forma parcial describe el comportamiento de los programas de computadoras que los procesan.

Es una aplicación del Standard Generalized Markup Language (ISO 8879) o SGML.

También se puede entender como un acuerdo entre personas para almacenar y compartir datos textuales de forma estándar.

XML fue desarrollado por el “XML Working Group” originalmente conocido como “SGML Editorial Review Group” bajo el auspicio del World Wide Web Consortium (W3C).

Reglas de sintaxis

Para ser un documento XML válido debe cumplir con las siguientes reglas.

  • Un documento XML debe tener el prólogo
  • Un documento XML debe tener un elemento raíz
  • Todos los elementos deben tener etiqueta de apertura y cierre
  • Las etiquetas son case sensitive
  • Los elementos deben estar anidados apropiadamente
  • Los valores de los atributos deben estar entre comillas
  • Los espacios en blanco son preservados en XML
  • Los finales de cada línea en XML solo contienen un “LF” a diferencia de Windows que agrega un “CR” antes (“CR+LF”).
